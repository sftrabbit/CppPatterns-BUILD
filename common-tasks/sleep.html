#include <chrono>
#include <thread>

int main()
{
	std::chrono::milliseconds sleepDuration(20);

	std::this_thread::sleep_for(sleepDuration);
}

This sample is licensed under the CC0 Public Domain Dedication.

Intent

Block the execution of a thread for a given amount of time.

Description

On line 6, we create a std::chrono::milliseconds object representing the number of milliseconds to sleep (other duration units may also be used). On line 8, the call to std::this_thread::sleep_for will block the execution of the current thread for at least the given duration.
